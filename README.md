# Suckling GitHub Pages Site

Static GitHub Pages minisite for the Return by 9 Discord bot, Suckling.

## Files

- `index.md` — homepage
- `COMMANDS.md` — public command reference
- `CHANGELOG.md` — release history
- `_layouts/default.html` — custom page shell
- `assets/css/style.css` — dark theme and typography
- `assets/img/suckling-logo.svg` — site logo

## Font

Headlines use Fraunces from Google Fonts, loaded in `_layouts/default.html`. No licensed font files are included.

## GitHub Pages

In the repo, go to Settings → Pages, then deploy from the `main` branch and `/root` folder.
